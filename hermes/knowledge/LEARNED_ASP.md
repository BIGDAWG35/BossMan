# ASP (Active Server Pages) Learning

## Overview
ASP = Active Server Pages
A development framework for building web pages
Server-side technology - code executes on Internet server

## ASP Models

### Classic ASP
- Introduced in 1998 (Microsoft's first server-side scripting)
- File extension: `.asp`
- Normally written in VBScript

### ASP.NET
- Released in 2002 as successor to Classic ASP
- File extension: `.aspx`
- Normally written in C#
- ASP.NET 4.6 is latest official version
- ASP.NET 5 development stopped in favor of ASP.NET Core

### ASP.NET Web Pages
- SPA application model (Single Page Application)
- Similar to PHP and Classic ASP
- Being merged into ASP.NET Core

### ASP.NET MVC
- MVC application model (Model-View-Controller)
- Being merged into ASP.NET Core
- Not covered in basic tutorials

### ASP.NET Web API
- API application model (Application Programming Interface)
- Being merged into ASP.NET Core
- Not covered in basic tutorials

### ASP.NET Web Forms
- Event-driven application model
- NOT part of ASP.NET Core

### ASP.NET Core
- Released in 2016
- Merges: ASP.NET MVC + ASP.NET Web API + ASP.NET Web Pages
- Into one unified application framework

## How It Works
1. Browser requests ASP/ASP.NET file
2. ASP engine reads the file
3. Executes any code in the file
4. Returns result to browser

## ASP.NET Core (Additional from Microsoft)

### Overview
- Free, cross-platform, open source
- Web development framework for .NET and C#
- Runs on macOS, Windows, Linux

### Key Features
- **Blazor** - web UI framework for creating fast, secure, robust web apps anywhere
- **Fast** - #1 in TechEmpower benchmarks
- **Secure** - Built-in authentication, MFA support
- **External auth** - Google, X (Twitter), and more
- **Protection** - XSS, CSRF protection

### E-Books (Free)
1. **Blazor e-book** - For Web Forms developers transitioning to Blazor
2. **ASP.NET Core architecture e-book** - Building monolithic web apps with Azure
3. **Azure quick start e-book** - Azure services for .NET apps
