# Transact-SQL (T-SQL) Learning

## Overview
Introduction to Transact-SQL - Microsoft's SQL dialect for relational databases.

## Module Details
- **Source**: Microsoft Learn
- **Units**: 9
- **Level**: Beginner

## Target Roles
- Data Analyst
- Data Engineer
- Data Scientist
- Database Administrator
- Developer
- Solution Architect
- Technology Manager
- Student

## Products Covered
- Azure SQL Database
- SQL Server
- Azure SQL Managed Instance
- SQL Server on Azure Virtual Machines
- Microsoft Fabric

## Learning Objectives

### After this module, you'll be able to:
1. Understand what SQL is for
2. Identify database objects in schemas
3. Identify SQL statement types
4. Use the SELECT statement
5. Work with data types
6. Handle NULLs
