# Digital Whiteboards Learning

## Overview
Digital whiteboards are collaborative tools for remote/hybrid teams to brainstorm, plan, and visualize ideas together in real-time.

## Types of Whiteboards

### Physical Whiteboards
- Markers, erasers, magnets
- In-person meetings
- Photo capture for sharing

### Digital Whiteboards
- Software for remote collaboration
- Cloud-synced boards

### Hybrid Setups
- Physical board with photo capture
- Cloud-synced boards

## Best Practices

### Before the Session
- Start with clear purpose and agenda
- Use templates (brainstorm, kanban, mind map, journey map, retro)
- Set up structure (sections, swimlanes, color-coding)

### During Collaboration
- Establish roles: facilitator, timekeeper, scribe, owner
- Use real-time cursors/avatars
- Set etiquette: mute when not talking, use chat for parallel thoughts
- Use voting/prioritization features

### Design Tips
- Use visuals: icons, shapes, arrows, connectors
- Keep it scannable: short phrases, consistent fonts
- High-contrast colors for accessibility
- Large fonts

## Quick-Start Plan

1. Pick 1 primary platform (Miro or Mural recommended)
2. Create starter board: agenda, problem statement, sections
3. Run 30-minute pilot with team
4. Collect feedback
5. Iterate on templates

## Considerations for Choosing

- Team size (small vs large)
- Remote vs in-person mix
- Required templates
- Integration needs (Slack, Teams, Google, Jira, Trello)
- Budget
- Security & permissions
- Offline access / mobile
