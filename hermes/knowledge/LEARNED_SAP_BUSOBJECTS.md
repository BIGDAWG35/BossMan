# SAP BusinessObjects (BOBJ) Learning

## Overview
SAP BusinessObjects is a business intelligence (BI) platform that provides reporting, analytics, and dashboarding capabilities.

## Core Components

### 1. BI Launch Pad
- Web-based interface for accessing BO content
- Repository for reports, dashboards, universes
- User authentication and session management

### 2. Web Intelligence (WebI)
- Ad-hoc reporting tool
- Query design and data analysis
- Create, view, and modify reports
- Key features:
  - Query design
  - Data providers
  - Blocks (tables)
  - Formulas
  - Prompts (parameters)
  - Contexts
  - Variables
  - Sorting, sections, formatting
  - Drill-down capabilities

### 3. Crystal Reports
- Professional report design tool
- Complex report layouts
- Subreports
- Formulas and formatting

### 4. Universes
- Semantic layer between users and data
- **Universe Design Tool (UDT)** - Legacy
- **Information Design Tool (IDT)** - Modern (UNX files)
- Components:
  - Joins
  - Classes
  - Objects
  - Prompts
  - Contexts
  - Slices
  - Security/Access control

### 5. Dashboards (formerly Xcelsius)
- Visual dashboard creation
- Data connectivity
- Interactivity
- SAP Analytics Cloud integration

## Architecture

### BOE/BI Platform
- **CMS** (Central Management Server) - Core server
- **Engine** - Processing requests
- **File Repository** - Stores objects
- **Security** - User authentication
- **Push Services** - Real-time updates
