## LLC Formation Details

### California LLC Forms
- **Articles of Organization**: Form LLC-1 (CA SOS)
- **Statement of Information**: Required
- **Operating Agreement**: Recommended (internal governance)
- **Form 568**: California LLC Return of Income (FTB)

### Fictitious Business Name (FBN)
- Required if using DBA name
- File with Orange County Clerk-Recorder
- Website: ocrecorder.com

### Name Search
- Check CA SOS for availability
- sos.ca.gov/business-programs/business-entities/
